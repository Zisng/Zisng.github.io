Spark-Pyspark

Spark-Pyspark基础编码环境

本机pyspark环境配置

  • 安装anaconda环境

  • 配置pycharm本地python解释器

  • 运行本地代码
    from pyspark import SparkConf, SparkContext
    import os
    os.environ[‘PYSPARK_PYTHON’]=’D:\Anaconda\anaconda\envs\pyspark\python.exe’
    if name == “main“:
    conf = SparkConf().setAppName(“WordCountHelloWorld”).setMaster(“local[*]”)
    sc = SparkContext(conf=conf)
    file_rdd = sc.textFile(“../data/input/words.txt”)
    words_rdd = file_rdd.flatMap(lambda line: line.split(“ “))
    words_with_one_rdd = words_rdd.map(lambda x: (x, 1))
    result_rdd = words_with_one_rdd.reduceByKey(lambda a, b: a + b)
    print(result_rdd.collect())

远程SSH Linux链接

在本地运行，可能会受到一些限制，而且在处理大量数据时，这些限制可能会影响pyspark运行效果，导致一系列性能问题。所以采用远程链接的方式来处理大大规模数据，用ssh远程方式运行pyspark。

  • 配置远程ssh linux解释器

    • 链接设置里的SSH Interpreter
    • 点击下图中的ok
    • 输入密码
    • 选择远程链接的解析器位置
      /export/server/anaconda3/envs/pyspark/bin/python3.8

  • 查看虚拟机上是否同步了pycharm中的文件

  • 运行ssh代码文件
    from pyspark import SparkConf, SparkContext
    import os
    os.environ[‘PYSPARK_PYTHON’]=’/export/server/anaconda3/envs/pyspark/bin/python3.8’
    if name == ‘main‘:
    conf = SparkConf().setAppName(“WordCountHelloWorld”).setMaster(“local[*]”)
    sc = SparkContext(conf=conf)
    file_rdd = sc.textFile(“file:///tmp/pycharm_project_96/data/input/words.txt”)
    words_rdd = file_rdd.flatMap(lambda line: line.split(“ “))
    words_with_one_rdd = words_rdd.map(lambda x: (x, 1))
    result_rdd = words_with_one_rdd.reduceByKey(lambda a, b: a + b)

hive3安装及应用

一、Mysql安装

  • 卸载Centos7自带的mariadb
    rpm -qa|grep mariadb
    rpm -e mariadb-libs-5.5.68-1.el7.x86_64 –nodeps
  • 安装mysql
    mkdir /export/server/mysql
    tar xvf my
    yum -y install libaio
    rpm -ivh mysql-community-common-5.7.29-1.el7.x86_64.rpm mysql-community-libs-5.7.29-1.el7.x86_64.rpm mysql-community-client-5.7.29-1.el7.x86_64.rpm mysql-community-server-5.7.29-1.el7.x86_64.rpm
  • 安装成功如下图
  • 初始化
    mysqld –initialize
    chown mysql:mysql /var/lib/mysql -R
  • 启动mysql
    systemctl start mysqld.service
    mysql -u root -p
  • 启动mysql成功如下图

二、Hive安装

  • 解压并设置软链接
    tar zxvf apache-hive-3.1.2-bin.tar.gz
    ln -s apache-hive-3.1.2-bin hive
  • 解决hive与hadoop之间guava版本差异
    cd /export/server/hive/
    cp /export/server/hadoop/share/hadoop/common/lib/guava-27.0-jre.jar ./lib/
    • 在hive目录下，删除其lib路径下的guava-19.0.jar，并将hadoop中的guava-27.0-jre.jar复制到hive的lib下
  • 修改配置文件
    cd /export/server/hive/conf
    mv hive-env.sh.template hive-env.sh
    vim hive-env.sh
    vim hive-site.xml
    文件尾部加入
    export HADOOP_HOME=/export/server/hadoop
    export HIVE_CONF_DIR=/export/server/hive/conf
    export HIVE_AUX_JARS_PATH=/export/server/hive/lib
    • 进入hive的conf路径下,移动hive-env.sh.template到hive-env.sh并编辑hive-env.sh和hive-site.xml文件
  • 初始化元数据
    cd /export/server/hive/
    bin/schematool -initSchema -dbType mysql -verbos
    • 初始化成功，会在mysql中创建74张表
  • 启动hive
    /export/server/hive/bin/hive –service metastore
    /export/server/hive/bin/hive –service metastore &
    /export/server/hive/bin/hive –service hiveserver2 &
    • 启动metastore服务,前台、后台和hiveserver2服务，启动hiveserver2时需要一定的时间
  • beeline客户端连接
    scp -r /export/server/apache-hive-3.1.2-bin/ root@node3:/export/server/
    /export/server/hive/bin/hive
    • 将node1的安装包拷贝到beeline客户端(node3)机器上并连接访问
